# personal-project-stockprice-prediction
Personal exploratory project on stock price prediction using Python-based modeling and historical market data.

## Tools used

Python

##  What is inside

- raw and processed stock price data (`.xlsx`)
- scripts to import, predict, and evaluate stock price models
- prediction charts and results

## Project contents

### Data files
- `stock_data_historical.xlsx` : historical information used in the project
- `CHARTS_FOR_PREDICTIONS_BIGTECH.xlsx` : visual results summarizing predicted vs. actual prices

### Python scripts
- `import_prices.py` : data from external sources
- `predict_new_batch32.py` : trains and runs prediction models
- `calculate_errors_rdeterminant.py` : calculates error metrics for prediction results
